### **1. Project Title / Headline**

* Super Store Sales Dashboard using Power BI
* Retail Sales Analysis Dashboard (Power BI)
* Interactive Business Intelligence Dashboard for Superstore Data
* Sales & Profit Insights Dashboard using Power BI

---

### **2. Short Description**

* Built an interactive dashboard to analyze retail sales and profit trends.
* Transformed raw data into meaningful insights for business decision-making.
* Identified top-performing regions, categories, and products.
* Enhanced data visualization using dynamic filters and KPIs.

---

### **3. Tech Stack**

* Power BI (Dashboard & Visualization)
* DAX (for calculated measures and KPIs)
* Power Query (for data cleaning and transformation)
* Excel / CSV (data handling)

---

### **4. Data Source**

* Superstore dataset obtained from Kaggle
* Publicly available retail dataset for analysis practice
* Data includes orders, sales, profit, customers, and product details
* Imported data from CSV/Excel into Power BI

---

### **5. Features**

* Interactive slicers for region, category, and date filtering
* KPI cards for total sales, profit, and quantity
* Trend analysis using line and bar charts
* Drill-down functionality for detailed insights
